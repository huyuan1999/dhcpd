这是一个专门为pxe/ipxe装机实现的dhcp服务器，它可能不满足任何的标准。

开发进度：开发中

想要实现的功能：
* dhcp 核心功能（包括分配IP，主机名，路由，网关，DNS等）
* 基于restful api的动态配置
* 基于数据库的mac地址绑定
* ipxe 的支持

目前完成的功能：
* dhcp 核心功能（大部分功能完成并测试成功）

未完成任务：
* 完成表结构设计
* 将租约信息存储到数据库中
* 将配置信息存储到数据库中
* 处理 Decline 请求
* 处理 Release 请求
* 编写处理释放IP相关操作
* 编写续租请求相关操作
